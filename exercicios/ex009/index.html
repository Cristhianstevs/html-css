Trabalhando com listas

Listas ordenadas (ol)

Essas listas geralmente são numeradas como padrão, isso acontece porque recebem uma ordem específica para serem seguidas, podem ser organizadas de diversar formas, como números romanos ou até mesmo letras. Para utilizar esse tipo de lista usa-se a tag ol.
Para criar as linhas de uma lista usa-se a tag li.
Para mudar o tipo de numeração escreva type="" dentro da tag ol, nas aspas escreva:

1 => Numeração
A => Letras maiúsculas
a => Letras minúsculas
I => Números romanos maiúsculas
i => Números romanos minúsculas

  1. Acordar
  2. Ligar para joão
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar pra casa
  9. Jantar
  10. Dormir

Listas não ordenadas (ul)

Essas listas não são necessariamente organizadas podendo ser colocadas em qualquer ordem. Esse tipo de lista usa símbulos para organizar.
Para utilizar esse tipo de lista usa-se a tag ul.
Para mudar o tipo de numeração escreva type="" dentro da tag ul, nas aspas escreva:

disc => Círculo padrão
circle => Círculo aberto
square => Quadrado

  • Pão
  • Leite
  • Tomate
  • Alface
  • Arroz
  • Feijão

Minhas linguagens favoritas

É possivel criar listas dentro de listas usando as tags logo após a linha. Exemplo:

  1. Antigas
    1. Clipper
    2. Visual Basic
    3. Fortran
    4. Delphi
  2. Novas
    1. PHP
    2. Python
    3. JavaScript
    4. Kotlin
