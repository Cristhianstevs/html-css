Lembretes

Os símbulos "< >" se chamam tags.

Use letras minúsculas em nomes de arquivos

Quando quiser colocar um texto ou um comando em uma tag, existe um atalho no qual você seleciona o texto ou palavra e aperta "Ctrl" + "Shift" + "P", aparecendo uma barra de pesquisa digite "abb" e selecione a opção "wrap with abbreviation" agora escreva a tag na qual o texto ou palavra que selecionou ficará entre

Títulos e subtítulos

Para criar títulos use a tag "h1", consequentemente, usando "h2" e "h3" podem ser criados subtítulos.
As tags "h" podem chegar até "h6" mas geralmente não é utilizado, alguns exemplos:

<H1>

<H2>

<H3>

parágrafos

Inserindo imagens

Para colocar uma imagem no seu site, primeiro ascolha uma imagem adequada e lembre de que ela precisa ser livre para uso, após escolher uma imagem livre, redefina a imagem para ter um tamanho adequado para o site, caso seja muito grande pode deixar o site lento. Jogue a imagem dentro da pasta na qual está o seu código.

Favicon

Para colocar um ícone no seu site primeiro escolha dentre vários na internet, lembre que precisa ser no formato ".ico". Coloque o arquivo dentro da pasta na qual está seu código.
Em "Head" escreva "link:favicon" e dê enter, apague o termo "favicon.ico" e dentro dessas aspas aperte Ctrl + Espaço, abrindo um atalho para a pasta na qual foi colocado o ícone e selecione o arquivo ".ico".
