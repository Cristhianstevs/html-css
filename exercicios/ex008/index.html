Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito "termo em negrito" usando a teg "b" (não semântica).

Nesta frase tem um "termo em destaque" usando a teg "Strong" (com semântica).

Itáloco / Ênfase

Nesta frase temos um "termo em itálico" usando a tag "i" (não semântica).

Nesta frase temos um "terno em itálico" usando a tag "EM" (com semântica).

Texto marcado

Podemos criar tambem um "texto marcado" usando a tag "MACK".

Temos outro "texto marcado" no final.

Texto grande e pequeno

Estamos criando um "texto grande" com a tag "big" (que já está obsoleta) e um "texto pequeno" com a tag "small" (que ainda não está obsoleta) neste parágrafo.

Texto deletado

Podemos criar um "texto como excluido" usando a tag "del" para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos criar um "texto como inserido" com a tag "ins" para dar uma ênfase e indicar que ele foi adicionado depois.

Existe tambem o "sublinhado" com a tag "u" (Não semântica).

Texto sobrescrito

Podemos inserir coisas do tipo "x20+3" com a tag "Sup" para deixar o numero 20 elevado.

Letras também funcionam, por exemplo "2x=8", usando a tag "sup" na letra x.

Texto subscrito

Podemos inserir coisas como "h2o" com a tag "sub" para deixar o número 2 em baixo.
