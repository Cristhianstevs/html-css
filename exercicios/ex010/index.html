Usando links externos

Para anexar um link a algum termo em html usa-se a tag a.
Para facilitar pode pegar um termo do texto, selecionar e apertar Ctrl + Shift + p, clicar em Wrap with abbreviation e colocar a tag a nele.

Você pode acessar o repositório do Gustavo.

Você também pode acessar o canal dele no YouTube. apenas clique aqui.

Para abrir o link em outra aba usa-se o comando target criando target="", dentro das aspas escreva _blank.

Curiosidade: Quando quiser entar em um link abrindo em uma nova janela, clique no link com o botão do Scroll do mouse.

Rel

(rel="")
O rel é uma questão de semântica para saber do que se trata esse link. Existe quatro tipos de rel, são eles:

  • next (próxima parte do documento atual)
  • prev (parte anterior do documento atual)
  • author (site do autor do artigo atual)
  • external (site que não faz parte do site)
